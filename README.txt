Repository containing configuration files which can be managed by using
GNU Stow.

How-to:
------

1. Clone the repository in your home directory.
2. Enter the directory.
3. Do 'stow ${FOLDER_NAME}'. This creates corresponding symlinks for the
   configuration files.

Please refer to GNU Stow Documentation for more options and details.
